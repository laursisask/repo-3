Filestorage - smart contract, that controls decentralized file storage on SKALE chains.

## Description

Filestorage is a cost-effective storage layer within Ethereum capable of handling files up to 100MB. Smart contract is a
set of functions to interact with files on SKALE chains. This contract is predeployed on each SKALE chain. It contains functions to 
work with precompiled smart contracts. Smart contract language - Solidity 0.4.24
